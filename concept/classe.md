#  Classe

Une classe a un nom.

Une classe est un ensemble d'étudiant.

Un ou plusieurs [cours](cours.md) peuvent être affecté à une classe par l'acteur [AdminLMS](../acteurs/adminLMS.md).

Une classe a un niveau: CM2, 6ème, Terminal, L2, M1
